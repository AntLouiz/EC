# EC
Project for manage students in English College.

### Requeriments

   - Python 3
   - Django 2

### Install VirtualEnv

```sh
$ virtualenv --python=python3 venv
$ source venv/bin/Activate
```

### For install Requeriments

```sh
$ pip install -r req_dev.txt
$ python manage.py migrate
$ python manage.py runserver
```
